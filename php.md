
# persiapan

- unduh PHP sesuai kebutuhan di https://windows.php.net/download/
- ekstrak di dalam `./` dengan penamaan folder `php` atau dengan penamaan menggunakan prefix `php`.

# kebutuhan extensions

- `curl`
- `intl`
- `gd` atau `gd2`
- `exif`
- `fileinfo`
- `mbstring`
- `openssl`
- `pgsql`
- `mysql`
- `mysqli`
- `sqlite3`
- `pdo`
- `pdo_odbc`
- `pdo_mysql`
- `pdo_pgsql`
- `pdo_sqlite`
